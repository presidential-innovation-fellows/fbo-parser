# FBO Parser

A parser for [fbo.gov's](https://www.fbo.gov/) daily FTP dumps.

`ftp://ftp.fbo.gov/` has the data.

## Usage

1. Grab the file(s) from `ftp://ftp.fbo.gov`
2. `var parser = require("index")`
3. `parser.parse(string)` returns valid JSON

## Example

Run `node example.js`
